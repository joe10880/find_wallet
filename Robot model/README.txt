# Summary

***Note: My origins are off so you will need to reorient some parts for optimal print quality.

Assembly video: https://youtu.be/N6g-pxyEQwk

This is a pan and tilt mechanism to protect your Pixy2 camera module. Uses 2 standard SC90 micro servos. 

If you just need a protective case there is plain back without mounting bracket.
